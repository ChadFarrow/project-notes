# MSP-2.0

**Category:** Music/Podcasting  
**Language:** TypeScript  
**Repo:** https://github.com/ChadFarrow/MSP-2.0

## Description
Music Side Project Studio - RSS feed generator for decentralized music releases

## PC2.0 Specs Used
- `<podcast:value>` - streaming sats
- `<podcast:valueRecipient>` - payment splits
- `<podcast:medium>` - music content type
- `<podcast:guid>` - unique identifier
