# Nostr References

## Specs
- [Nostr Protocol](https://github.com/nostr-protocol/nostr) - Core protocol
  - Docs: https://nostr.com/
- [NIPs](https://github.com/nostr-protocol/nips) - Nostr Implementation Possibilities
  - Docs: https://github.com/nostr-protocol/nips
- [NIP-07](https://github.com/nostr-protocol/nips/blob/master/07.md) - Browser extension API
- [NIP-46](https://github.com/nostr-protocol/nips/blob/master/46.md) - Remote signer protocol
- [NIP-47](https://github.com/nostr-protocol/nips/blob/master/47.md) - Nostr Wallet Connect
