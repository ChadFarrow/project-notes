# Lightning Network References

## Specs & Protocols
- [LNURL Specs](https://github.com/lnurl/luds) - Lightning URL specifications
  - Docs: https://github.com/lnurl/luds
- [Lightning Address](https://lightningaddress.com/) - Email-like LN addresses
  - Docs: https://lightningaddress.com/
- [BOLT Specs](https://github.com/lightning/bolts) - Lightning protocol specs
  - Docs: https://github.com/lightning/bolts
- [NIP-47](https://github.com/nostr-protocol/nips/blob/master/47.md) - Nostr Wallet Connect spec
  - Docs: https://github.com/nostr-protocol/nips/blob/master/47.md
- [bech32](https://github.com/bitcoinjs/bech32) - Bech32 encoding/decoding
  - Docs: https://www.npmjs.com/package/bech32

## Wallets & Services
- [Alby](https://getalby.com/) - Lightning browser extension wallet
  - Docs: https://guides.getalby.com/
- [Mutiny Wallet](https://www.mutinywallet.com/) - NWC-compatible wallet
  - Docs: https://www.mutinywallet.com/
- [coinos-server](https://github.com/coinos/coinos-server) - Coinos backend server
  - Docs: https://coinos.io/docs
- [Lightning.Pub](https://github.com/ArkLabsHQ/Lightning.Pub) - Nostr native Lightning node
  - Docs: https://lightning.pub/

## Libraries & SDKs
- [bitcoin-connect](https://github.com/getAlby/bitcoin-connect) - Alby Bitcoin Connect
  - Docs: https://bitcoin-connect.com/
- [webln-types](https://github.com/webbtc/webln-types) - WebLN type definitions
  - Docs: https://www.webln.dev/
- [spark-sdk](https://github.com/ArkLabsHQ/spark-sdk) - Rust SDK for Lightning
- [breez-sdk](https://github.com/breez/breez-sdk) - Breez Lightning SDK
  - Docs: https://sdk-doc.breez.technology/
- [lightning-address-details-proxy](https://github.com/getAlby/lightning-address-details-proxy) - Lightning address proxy
