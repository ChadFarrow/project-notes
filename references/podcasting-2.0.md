## Apps & Tools
- [Podcast Index](https://podcastindex.org/) - Search engine
  - Docs: https://podcastindex.org/developers
- [Podping](https://podping.org/) - Real-time feed notifications
  - Docs: https://podping.org/
- [podstr](https://github.com/nostr-protocol/podstr) - Nostr + Podcasting 2.0 app
- [pimonitor](https://github.com/Podcastindex-org/pimonitor) - Monitors incoming feeds to Podcast Index API
- [feedparser](https://github.com/Podcastindex-org/feedparser) - XML parser for podcast feeds
- [sovereign-feeds](https://github.com/sovereign-feeds/sovereign-feeds) - Self-sovereign podcast feeds
- [live-updates](https://github.com/Podcastindex-org/live-updates) - Live updates for podcasts
